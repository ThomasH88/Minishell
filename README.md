# Minishell
Fully functional minimal bash shell

### Currently Implemented
- echo (minimal)
- cd
- setenv
- unsetenv
- env
- exit
- expansions $ and ~
- commands separated by ;
## How to run
Clone the repo and run the executable:
```bash
git clone https://github.com/ThomasH88/Minishell.git
./minishell
```
